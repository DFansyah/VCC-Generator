# Installation
## For Termux User

```
pkg update
pkg upgrade
pkg install nodejs
pkg install npm
pkg install git
termux-termux-storage
git clone https://github.com/DFansyah/VCC-Generator
cd VCC-Generator
npm install
npm start or node index.js
```

If you can install Packages on npm you can run this commands

```
pkg install yarn
yarn install
npm start or node index.js
```
